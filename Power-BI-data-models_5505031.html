Introduction

On top of the data warehouse a specific view is created for power bi. In this view some last transformations are added for performance reasons or for reporting visualizations. This layer is called the pbi_datamodel and forms the basis for the power bi data model which you see and use via power bi desktop.

The model has:

  • fact tables for measures

  • Dimension tables to answer questions about the measures. like when, where, who etc.

  • The model uses meaningful names

Structure of the power bi data model

A power bi data model is a combination of measures and dimensions and hierarchies.

The measures tells about the what: number of activities. while the dimensions tells more about when, where, who etc. The hierarchy let you drill down in visualizations.
