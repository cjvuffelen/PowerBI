IT Documentation : Add/ remove people from Automated Code Reviewers in Github

Created by r.lodder1, last modified on jul. 08, 2022

For MuleSoft APIs, we use automated assignment of persons to a pull request. This works hand in hand with Setup CI/CD in GitHub Team .

 Instructions

The steps to take:

  1. Follow Setup CI/CD in GitHub Team and head to ‘Code Review’

  2. In our Team the option ‘Never assign certain team members’ is checked. This means to can exclude certain members of the team from being assigned. Select/deselect the person(s) and Save changes.
