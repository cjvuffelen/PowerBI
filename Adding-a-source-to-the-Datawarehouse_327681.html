Weird ADF errors:

Error: Pipeline will not run without any error. → Possible cause → the parameters in a dataset are empty. When you add them and save the are not added. → restart ADF → open the dataset directly and try again.

Error: Validation error for stuff that doesnt exist anymore. The problem is that he parameter exists in the github repository. They are out of sync. Click on the validation error, drill down in the activities and validate the lower pipelines. ADF will then see it is no longer correct and the error is solved.
