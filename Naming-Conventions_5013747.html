Anypoint MQ  

Destinations  

Dot separated, upper case 

{E|Q|DLQ|RETRYQ}.{source?}.{domain?}.{entity}.{event?}.{message format?}.{consuming api/app?}.{target?}.{env?}.{version?} 

 

E.CUSTOMER 

Q.IFS.ISSUE.CDM 

 

Destination types:  

E = exchange 

Q = queue 

DLQ = dead letter queue 

RETRYQ = retry queue 

 

The dead letter queue has the same identifiers as the main queue.
