All you need to know about custom visuals

There is a short list you need to know about custom visuals:

  1. Limit the use, where it adds business value

  2. They can be extremely helpful in smooth your reports (visualize progress on pictures of boats).

  3. They can be downloaded from marketplace

  4. Only certified visuals are allowed

  5. There is a potential security with custom visuals not certified

Security and un-certified custom visuals

As there is a security risk, IHC only wants certified visuals to be added and used in the organization. In case any custom visual not certified needs specific action by admin’s to be used. Request this via CDM.

Custom visuals not certified are not loaded into the service except for those which meet the following conditions:

  1. There is no alternative available

  2. The visual adds business value

  3. The custom visual is validated, and documented by CDM

  4. The custom visual is checked by IHC security officer.

Requests for custom visuals

For any request related to custom visuals like adding it to the service or allowing un-certified visual to be accepted a ticket should be created to CDM describing the need and use case. The request can be either send directly to one of the CDM team members or via Data backlog in Jira: https://royalihc.atlassian.net/jira/software/projects/DATA/boards/92/backlog
