IT Documentation : Contacts API

Created by Chi Chuen Ng, last modified on mrt. 31, 2022

The contacts API retrieves data from IFS and returns it to the client. It is part of the MARS-DMS interface and DMS uses it to get contacts and contact addresses. Please click here for more information about the Contacts System API spec.

Contacts

The API gets contacts by calling stored procedure IFSAPP.IHC_PORTAL_UTIL_API.get_contacts(:cv_1,:cont_id_,:last_modified_,:ihc_project_id_) where cv_1 is a output cursor and the rest are input variables. The last_modified_ variable expects a datetime format of “yyyyMMddHHmmss“. All variables are optional, however, the Contacts System API spec requirements are different.

The API spec expects projectId, lastModifiedDateTime (format: “yyyy-MM-dd’T’HH:mm:ss’Z'“) and contactId queryParameters of which the first two are required. ContactId is an optional parameter.

The flow itself is simple: a stored procedure call is made and the output is transformed into an JSON response.
