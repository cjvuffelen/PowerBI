IT Documentation : ZScaler certificate in pipenv environment

Created by ewoud westerbaan, last modified on apr. 06, 2022

Infastructure of IHC uses ZScaler for secure networking. For dbt, we use a python environment. However, python does not use the default certificate store of windows. Therefore we need to manually add the ZScaler root certificat to the python environment.

  1. Go to ‘Manage computer certificates’

  2. Go to 'Action → Find Certificates'

  3. Search for ‘ZScaler’

  4. Double click certificate

  5. Go to ‘Details → Copy to file’

  6. Click Next

  7. Choose Base-64 encoded X.509 (.CER).

  8. Click Next.

  9. Choose a location.

  10. Open the saved file in Notepad.

  11. In your pipenv, go to Lib\site-packages\certifi. You can find the location by clicking in the python version in the left bottom of VSCode (or in your environment, type: pipenv --venv).

  12. Open cacert.pem

  13. Copy the content of the saved certificate to the bottom of the cacert.pem file.
